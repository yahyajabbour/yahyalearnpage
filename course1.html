HTML Course

HTML (HyperText Markup Language) est le langage de base du web. Il permet de structurer le contenu d’une page.

1. Structure de base

<!DOCTYPE html>
<html>
  <head>
    <title>Titre de la page</title>
  </head>
  <body>
    <h1>Bonjour</h1>
    <p>Ceci est un paragraphe.</p>
  </body>
</html>

2. Les balises courantes

  • <h1> à <h6> – Titres
  • <p> – Paragraphe
  • <a href=""> – Lien
  • <img src=""> – Image
  • <ul>, <ol>, <li> – Listes

3. Exercice simple

Crée une page HTML avec un titre, une image et une liste de tes hobbies.
