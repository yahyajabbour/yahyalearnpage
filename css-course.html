CSS Course

CSS (Cascading Style Sheets) est un langage qui permet de styliser les pages HTML : couleurs, tailles, marges, disposition, etc.

1. Ajouter du CSS

  • Interne : dans une balise <style>
  • Externe : via un fichier .css
  • Inline : directement dans les balises HTML

2. Exemple de style

h1 {
  color: brown;
  font-size: 2em;
}
p {
  background-color: #f1f1f1;
  padding: 10px;
}

3. Sélecteurs

  • element → p
  • #id → #main-title
  • .class → .card
