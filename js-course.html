JavaScript Course

JavaScript est un langage de programmation qui permet de rendre les pages web interactives : clics, formulaires, animations, etc.

1. Inclure du JavaScript

<script>
  alert('Bienvenue !');
</script>

2. Concepts de base

  • Variables : let x = 5;
  • Conditions : if (x > 0) { ... }
  • Événements : onclick, onchange

3. Exemple d'interaction

<button onclick="alert('Salut !')">Clique-moi</button>

4. Exercice

Crée une page avec un bouton qui affiche une alerte quand on clique dessus.
